# My One Piece TCG Tracker

This is a personal project I built to **design, save, and manage my One Piece TCG decks**, while also tracking which cards I physically own and which ones I still need to collect[cite: 1].

Initially, the app saved everything directly in the browser (via LocalStorage), but to avoid losing data and to keep my decks secure, I connected it to a **MongoDB** database using a small **Node.js** server[cite: 1].
